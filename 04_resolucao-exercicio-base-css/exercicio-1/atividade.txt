Utilizando CSS inline aplique as seguintes regras CSS:

1. Adicione uma cor de fundo com o valor do tipo Hexadecimal no corpo da página;

2. O título principal da página (h1) deve estar em caixa alta (maiúsculo). Utilize CSS inline para aplicar a transformação;

3. Nos títulos de cada seção (Formação, Experiências Profissionais e Contato), aplique:
- Um tamanho de fonte de 24px;
- Uma cor de fundo com o valor do tipo RGB;

4. No parágrafo do rodapé da página, aplique:
- Um tamanho de fonte de 14px;
- Uma cor de texto com o valor do tipo Nome para qualquer tom de cinza;
- Alinhe o texto ao centro da página;
